# facef-pos-testes
Projeto de demonstração quanto a aplicação para a prática de testes automatizados - unitários e simulação de mock de dependências.

## Setup
* java 11
* maven

## Execução
`mvn compile`

## Testes
`mvn test`

## Geração jar
`mvn clean package`

## Gerar imagem
`docker build -t nome:version .`
